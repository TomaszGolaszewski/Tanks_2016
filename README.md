## Instruction
- To play 'Tanks', download and unpack _Tanks Alfa 0.1.9.zip_ archive, then run the file _Tanks.exe_
- Controls: 
  - Player I: movement: WASD; tower rotation: FH; fire: G
  - Player II: arrows; tower rotation: ,(<) .(>) ; fire: /
